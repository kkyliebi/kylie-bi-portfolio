## Supported project identity

### Working title

**TYRANNO**

### Supported project description

A speculative communication-design project that imagines human and Tyrannosaurus rex coexistence in an urban 2050 scenario. The project develops a behaviour-driven symbolic language and typeface system for interspecies communication.

### Supported domains

- speculative design
- semiotics
- typography and visual language
- paleontology and T. rex behaviour
- interaction design

## Six-chapter evidence map

### 1. Context

#### Supported

- The project is set in a speculative 2050 context.
- The scenario imagines humans and T. rex living in proximity after de-extinction or biological reconstruction.
- The communication problem begins with the limitations of human language when addressing a non-human species with different anatomy, perception and behaviour.
- The project asks how a communication system might be designed without assuming speech as the primary channel.

- gesture and non-verbal communication

- speculative design and practice-based design research

### 3. Concept Development

#### Supported

- The central concept translates observed or inferred T. rex behaviours into a human-readable symbolic system.
- The system is behaviour-driven rather than phonetic.
- Motion, posture and gesture are treated as communication material.
- The project explores how icons, indexes and symbols can work together across species.
- The symbolic language develops from a smaller core set into a broader system.
