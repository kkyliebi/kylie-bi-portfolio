All copy described here is **Draft** unless explicitly identified as existing metadata. Drafts are generated in `src/content/createPlaceholderNarrative.ts` from each project's structured metadata. They are public-facing starting points, not verified project records.

## Shared evidence

The repository establishes a common case-study sequence: Context, Research, Concept Development, Process, Outcome, Reflection. The master brief establishes the method—Discover, Connect, Translate, Materialise—and the principle that each project demonstrates how abstract ideas become communication systems.

This supports drafting each chapter's narrative role. It does not establish historical facts about a project's brief, client, collaborators, chronology, research findings, production techniques, deliverables, reception, or impact.
