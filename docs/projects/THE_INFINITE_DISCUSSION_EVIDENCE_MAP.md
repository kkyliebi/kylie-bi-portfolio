## Provisional publication strategy

For the minimum job-application release, use one of three paths:

1. **Recover and verify** — retain the project and build the full case from original files;
2. **Keep as a concise archive entry** — show only verified title, role, format and selected outcome media;
3. **Temporarily withhold** — keep the route configured internally but remove it from the public archive until evidence is sufficient.

The second option is preferable if strong final visuals exist but process evidence is limited. The third is preferable if neither factual records nor final media can be recovered quickly.
