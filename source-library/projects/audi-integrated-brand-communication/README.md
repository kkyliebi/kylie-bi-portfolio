## Files

- `canonical.md` — maintained project source of truth
- `CHANGELOG.md` — version history and editorial decisions
- `working/` — supporting notes, responsibility records, and intermediate documents
- `original/` — preserved source files
- `assets/raw/` — unsorted visual material before editorial selection

## Portfolio scope

This project is treated as one integrated brand communication case for Portfolio V1. It may later contain distinct sub-cases such as Audi Channel, Audi A8L Horch Global Premiere, and Audi × Ryuichi Sakamoto, but no additional public-facing subdivision should be assumed until the editorial structure is confirmed.

## Working rule

Only verified responsibilities, outcomes, and evidence should enter `canonical.md`. Unverified metrics, role inflation, and unsupported claims must remain excluded.
