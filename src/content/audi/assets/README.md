# Audi Asset Convention

Use this folder for production assets when available.

- `hero-*`: project entry imagery
- `research-*`: source material, references, research grids
- `process-*`: sketches, iterations, development sequences
- `gallery-*`: final visuals and outcome spreads
- `archive-*`: supporting documents and contextual material

Every asset should have matching content metadata with `caption` and `alt` text.
