# The Infinite Discussion Source Library

## Status

- Canonical content: pending import from confirmed project materials
- Recent discussion update: pending comparison and selective revision
- Raw visual assets: upload to `assets/raw/`

## Files

- `canonical.md` — maintained project source of truth
- `CHANGELOG.md` — version history and update rationale
- `working/` — plans, role notes, drafts, and intermediate documents
- `original/` — source PDFs and other preserved originals
- `assets/raw/` — unsorted visual material before editorial selection

## Working rule

Existing confirmed content is preserved by default. New discussion only changes sections where understanding has genuinely improved.
